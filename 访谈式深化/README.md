# 访谈式深化

> 通过访谈激发思考，帮助内容创作者把脑子里模糊的想法，变成听起来是自己说的高质量内容。

---

## 产品定位

**核心理念：** 你负责思考和口述，我负责提问和整理。最终产出听起来是你说的，不是AI写的。

**适合人群：**
- 有想法但说不清楚
- 用AI写出来的东西不像自己
- 面对空白页不知道从哪开始

**核心能力：**
1. **聚焦策略** — 帮你找到最值得讲的那个点
2. **动态访谈** — 根据你的回答灵活调整提问方向
3. **风格保留** — 保留你的说话方式，不改成书面语
4. **版本管理** — 支持多版本迭代，随时回溯
5. **风格档案** — 越用越懂你的表达习惯

## 使用方式

### 方式1：Claude Code Skill（推荐）

如果你使用 Claude Code，可以将产品版安装为Skill：

1. 下载 `访谈式深化SKILL-产品版.md`

3. 重命名为 `SKILL.md`

**优势：**
- 无需安装，即复制即用
- 适用于所有AI工具

## 工作流程

1. **了解想法** — AI问你想做什么内容
2. **确认目标** — 明确商业目标（引流/建立信任/转化）
3. **动态访谈** — 一次一个问题，根据回答灵活调整
4. **整理成文** — 保留你的表达方式，按逻辑重组
5. **口播顺稿** — 让句子像人嘴里说出来的
6. **版本管理** — 支持多版本迭代
7. **定稿流转** — 草稿 → 已定稿 → 已发布
8. **风格更新** — 自动记录你的表达习惯

## 核心特性

### 聚焦策略

如果选题有多个并列点，AI会先问"哪个最有实战感"，集中挖透一个最强的点。

### 动态访谈

根据你的回答情况灵活调整提问：
- 回答太浅 → 追问具体案例
- 回答太抽象 → 要求用比喻解释
- 回答太散 → 帮你聚焦核心观点
- 有意思的点 → 深入展开

### 风格保留

- 保留你的原话和表达方式
- 不用书面语改写
- 你说"这玩意儿"就保留"这玩意儿"
- 参照风格档案，回避你排斥的表达

### 发布包装

自动生成：
- **封面文案** — 2-4行，让人一眼停住
- **小红书标题** — 20字内，有反差或悬念
- **抖音标题** — 讲清因果，口语化

## 常见问题

**Q：这跟直接让AI写有什么区别？**

A：直接让AI写，写出来是AI的表达方式。用访谈式深化，AI负责追问和整理，内容是你自己说出来的，听起来才像你。

**Q：我说话比较乱，AI能整理好吗？**

A：可以。AI遇到不通顺的句子会先确认你的意思，不会自己猜测乱改，整理完保留你说话的方式。

**Q：需要什么基础吗？**

A：不需要。只要你有想法（哪怕很模糊），AI会通过提问帮你理清思路。
