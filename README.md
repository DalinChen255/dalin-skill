## 产品列表

### 访谈式深化

帮助内容创作者通过结构化访谈，将模糊想法深化为完整的内容框架。

**适用场景：**
- 有想法但不知道怎么展开
- 想做内容但不知道讲什么
- 需要快速验证选题价值

**核心能力：**
- 聚焦策略：帮你找到最值得讲的角度
- 结构化访谈：通过提问引导你说出隐性知识
- 风格检查：确保内容符合易中天式四段论叙事框架

**使用方法：**

查看 [访谈式深化](./访谈式深化/) 目录获取完整文档。
